2. 【重要】ドメイン認証の設定

GitHub Pagesなどの外部URLで動かす場合、Firebaseコンソールで許可設定が必要です：

  1. Firebaseコンソール ＞ Authentication ＞ Settings ＞ 「承認済みドメイン」 を開く
  2. 「ドメインの追加」を押し、GitHub Pagesのドメイン（例: あなたの名前.github.io）を入力して保存してください。
